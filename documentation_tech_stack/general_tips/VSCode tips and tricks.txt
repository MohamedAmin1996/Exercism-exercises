===================================================
WINDOWS 11 PRO - NORDIC/SWEDISH KEYBOARD SETUP LOG
===================================================

Each Language setup
-------------------
Engine or SDK
Language Extension
Formatter Extension + Setting up the formatter in User Setting JSON
Code Snippet Extension (For Python only, the rest has it by default) + prioritizing the code snippet in the setting json
IntelliSense (Is included when you install the extension)
Debug (JavaScript/TypeScript needs a fix, look below, rest work by default)
Unittest Extension (For JavaScript/TypeScript it is called ViTest. Install the package via CLI and the extension)

Migrations (C# only)
Building, Publish, Clean


JavaScript/TypeScript debug cleanliness
-------------------
To clean up the debug, add a file called 
package.json and add this to make sure the debug is as clean
as possible:

{
	"type": "module"
}

Core Settings
-------------
Open Settings UI:      CTRL + ,
Open Command Palette:  CTRL + SHIFT + P

Formatter + code snippet Configuration
-----------------------
Command Palette -> "Open User Settings (JSON)"
Add the following block at the root level:

  "[csharp]": {
    "editor.defaultFormatter": "csharpier.csharpier-vscode",
    "editor.formatOnSave": true
  }
 "editor.snippetSuggestions": "top"

Autocomplete Engine
-------------------
1. Type a keyword (e.g., 'for').
2. Press TAB to automatically expand snippets (if configured to "top").
3. Press ESC to completely dismiss the suggestion box.

IntelliSense & Methods
----------------------
See code documentation:     CTRL + SPACE
Show Method Overloads:      CTRL + SHIFT + SPACE (Inside parentheses)

Panel Toggle Shortcuts
----------------------
Problems Window:   CTRL + SHIFT + M
Output Logs:       CTRL + SHIFT + U 
Debug Console:     CTRL + SHIFT + Y
Terminal Panel:    CTRL + ` (or custom CTRL + SHIFT + Ö)

Integrated Debugger
-------------------
Start/Continue:    F5  (or FN + F5)
Step Over line:    F10 (or FN + F10)
Step Into method:  F11 (or FN + F11)
Step Out of method: SHIFT + F11 (or SHIFT + FN + F11)
Restart Session:   CTRL + SHIFT + F5
Stop Debugging:    SHIFT + F5

Unit Testing Suite
------------------
1. Click the Testing Sidebar icon (Science Flask/Beaker icon).
2. Press the Play button to execute the test suite automation.

NuGet Package Management (C#)
------------------------
1. CTRL + SHIFT + P -> Type "NuGet: Add Package"
2. Search and select package version to auto-install directly to project.

EF Core Migrations Workflow (C#)
---------------------------
Create structural blueprint:  dotnet ef migrations add InitialCreate --project DataApp
Push schema live to server:   dotnet ef database update --project DataApp

Building & Maintenance
----------------------
Via UI:       Right-click individual .csproj configuration files.
Via Terminal: dotnet build projectFolder/file.csproj
Clean Cache:  dotnet clean projectFolder/file.csproj

Dependencies Restoration
------------------------
Restore App:  dotnet restore
Targeted:     dotnet restore projectFolder/file.csproj

SQL Server Rules
----------------
1. Ensure the local MSSQL system service is turned on.
2. Establish database hook inside VS Code server dashboard explorer.
3. GOLDEN RULE: Never change tables manually with SQL queries. 
   Change the C# code first, then run a migration to push changes down.
